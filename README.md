## Sobre el proyecto

#### Instalar dependendencias con:
```
npm install --save-dev babel-cli babel-preset-env jest supertest
```

Para poder usar ECMA SCRIPT 6 incluiremos estas dependencias:
```
{
  "devDependencies": {
    "babel-jest": "^20.0.3",
    "babel-polyfill": "^6.26.0",
    "babel-preset-es2015": "^6.24.1"
  }
}
```
